

Sistema simples de gerenciamento de lista de convidados desenvolvido em JavaScript.

Praticar lógica de programação e manipulação de arrays criando um menu interativo que permite:

Adicionar convidados
Buscar convidados pelo nome
Remover convidados
Listar todos os convidados
Sair do sistema



Este projeto foi desenvolvido utilizando:

- Arrays
- push()
- indexOf()
- splice()
- join()
- Estrutura de repetição `do...while`
- Estrutura de decisão `switch`
- prompt() para interação com o usuário

Como Executar

1. Abra o arquivo `index.html` no navegador  
ou  
2. Execute o código em um ambiente que suporte `prompt()` (como o navegador)

Sobre

Projeto desenvolvido como atividade escolar para praticar fundamentos de JavaScript.
